# Lumina Scholar
An AI-powered academic assistant for students. Features PDF-contextual chat and study tools like quizzes and flashcards.

## Features
- **General & PDF Chat:** Ask anything! Upload PDFs to get context-specific answers.
- **Multilingual Support:** Responds in Turkish or English automatically based on your input.
- **Study Tools:** Generate custom Quizzes and Flashcards for any document.
- **Privacy First:** All data is processed in your browser using your own Gemini API Key.

## How to use
1. Visit the [Lumina Scholar Website](https://seray-d.github.io/lumina-scholar/)
2. Open Settings and enter your Google Gemini API Key.
3. Upload your PDF and start studying!
